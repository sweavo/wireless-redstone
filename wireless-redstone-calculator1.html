Wireless Redstone Calculator

Instructions

Enter redstone chains, one per line. Each chain consists of repeaters (R0-R3) and comparators (C).

Format: Each line contains combinations of:

  • R0, R1, R2, R3 - Repeaters with delays 0-3
  • C - Comparators
Example input:

R2R1
